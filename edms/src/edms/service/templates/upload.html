{# ── Side panel: what happens after upload ─────────────── #}

What happens next

  1. Stored in MinIO. File is uploaded to the bucket configured on the chosen collection.
  2. Text extraction. On first read, Vault extracts text from PDFs and other supported types and stores the result as a lineage child of the original.
  3. Tag governance. Tags are validated against the controlled vocabulary; required tags must be supplied or the upload fails.
  4. Available to Verity. Once stored, governed Verity tasks and agents can fetch the document via the edms data connector (e.g. classifier vision blocks, extractor text fetches).
