{# Save-status partial. Replaces the #save-status div under any Studio editor form. The HTMX ``hx-swap="outerHTML"`` swaps this whole div in, so the partial must include the wrapper element with the same id. Inputs (one of three shapes, by ``status``): status="ok": new_updated_at — ISO 8601 timestamp of the row's new stamp. entity_label — Short label for the saved entity to surface in the message (e.g. ``v1.2.0`` for a prompt version, ``claude_default`` for a config). Optional. status="conflict": current_updated_at — server's current stamp; informs the user how stale they were. reload_url — Absolute URL to GET to see the current state. The user's edits stay in the form above. status="error": message — free-form error message to surface verbatim. #}
