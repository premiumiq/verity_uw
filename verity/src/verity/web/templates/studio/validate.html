Validate is where a draft is exercised before promotion. Authors run live previews against any available model, execute validation batches against ground-truth datasets, and attach the resulting validation_run as evidence on a promotion request. Preview runs are tagged experimental and excluded from production telemetry.

The Validate workspace will go here. Expect a preview pane with schema-aware input forms, a mock-vs-live toggle, an ephemeral LLM override, and a batch runner with per-case drill-down.
