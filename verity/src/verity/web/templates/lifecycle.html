About Lifecycle Management
Every version of every agent and task moves through Verity's 7-state promotion sequence: draft → candidate → staging → shadow → challenger → champion → deprecated. Each transition requires evidence (tests passed, validation passed, model card reviewed) and an approver. This page shows all versions across all entities with their current state and promotion readiness.
