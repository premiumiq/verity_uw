About Validation
Validation runs test an entity version against a ground truth dataset. The runner executes the entity for every record, compares outputs to authoritative SME annotations, and computes aggregate metrics (precision, recall, F1, Cohen's kappa for classification; per-field accuracy for extraction). Results are checked against metric thresholds — all thresholds must be met for the version to pass. Validation gates staging → champion promotion.
