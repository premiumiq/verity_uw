About Ground Truth
Ground truth datasets contain production-representative data labeled by subject matter experts (SMEs). Each record has an input (a document, a submission context) and an authoritative annotation (the correct answer). The validation runner executes an entity version against every record and compares outputs to annotations, computing aggregate metrics (F1, precision, recall, Cohen's kappa). Ground truth validation gates staging → champion promotion.

Silver tier: Single annotator — one SME labels each record. Suitable for medium-materiality tasks.
Gold tier: Multi-annotator with inter-annotator agreement (IAA) check. Disagreements adjudicated by senior SME. Required for high-materiality agents.
