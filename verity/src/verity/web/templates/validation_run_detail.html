Metric Definitions
MetricDefinition
PrecisionOf all items labeled as class X, what fraction actually were X? High precision = few false positives.
RecallOf all items that actually were X, what fraction did the model identify? High recall = few false negatives.
F1 ScoreHarmonic mean of precision and recall. Balances both. Range 0-1, higher is better.
Cohen's KappaAgreement between model and SME adjusted for chance. 0 = random, 1 = perfect. >0.8 is strong agreement.
Field AccuracyFor extraction: fraction of fields where extracted value matches expected (within tolerance).
