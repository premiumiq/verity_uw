# Evaluation Run

> **Tooltip:** Lighter-weight scoring of an entity against test cases or selected ground-truth slices; not a promotion gate.

## Definition

A scored execution similar to a validation_run but not bound to a promotion gate. Used for ad-hoc quality checks, A/B comparisons, and dashboard reporting.
