# Incident

> **Tooltip:** Production triage row: legacy incidents + active quota breaches. Surfaces in the unified Incidents page.

## Definition

A row in `incident` (or a derived view) representing something in production that needs triage attention. Today combines explicitly-created incidents and active quota breaches. Future Verity Agents (drift detection) will create incidents automatically.
