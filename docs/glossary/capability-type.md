# Capability Type

> **Tooltip:** Entity classifier (classification/extraction/judgment/generation/matching/validation) shaping default validation metrics.

## Definition

An enum field on Agent and Task entities indicating what kind of work the entity performs. Drives default metric choices in validation runs (e.g. F1 for classification, per-field accuracy for extraction, exact-match for matching).
