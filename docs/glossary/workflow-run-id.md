# Workflow Run ID

> **Tooltip:** Caller-supplied UUID threaded through every execute_* call in one workflow so the audit clusters correctly.

## Definition

A correlation UUID generated by the consuming app, threaded through every Task or Agent invocation that belongs to one workflow execution. Verity treats it as opaque metadata. Recorded on every decision_log row so audit queries can return all related decisions in a single query.
