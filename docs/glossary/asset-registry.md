# Asset Registry

> **Tooltip:** Verity Governance subsystem storing every governed entity as a versioned database record.

## Definition

The Verity Governance subsystem responsible for storing, versioning, and resolving every governed AI component. Supports champion resolution (default current, date-pinned, or version-pinned) and clone-and-edit authoring with `cloned_from_version_id` lineage.
